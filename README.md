# 🧠 MindEase AI — Mental Health Support Chatbot

MindEase AI is a full-stack, AI-powered mental health support chatbot designed to provide empathetic conversations, mood tracking, guided exercises, and intelligent crisis detection — all within a clean, user-friendly chat interface.

---

## 🚀 Live Demo

🌐 https://mindease-ai-chatbot.vercel.app

---

## 📌 Features

### 💬 AI Chat Support

* Real-time conversational AI responses
* Empathetic, therapist-style interaction
* Context-aware replies with session tracking

---

### 😊 Mood Tracking

* Users can log their mood (Happy, Neutral, Sad, Anxious)
* Helps track emotional patterns over time
* Encourages self-awareness and reflection

---

### ➕ Guided Exercises & Coping Tools

* Breathing exercises
* Grounding techniques (5-4-3-2-1 method)
* Gratitude prompts
* **Smart Coping Tools Button**

  * When users ask for help or alternatives, the system can guide them toward exercises and calming techniques

---

### 🚨 Crisis Detection (Enhanced)

* Detects high-risk phrases in real-time
* Triggers supportive responses
* Displays helpline access when needed
* Designed to provide safer interactions during vulnerable moments

---

### 🔒 Anti-Spam UX

* Prevents message flooding while AI is responding
* Improves conversational flow and realism

---

### 🌙 Dark / Light Mode

* Toggle between light and dark themes
* User preference saved locally

---

### 🧠 AI Memory (Session-Based)

* Maintains context within a conversation session
* Enables more natural and continuous interactions
* Simulates short-term conversational memory

---

## 🛠️ Tech Stack

### Frontend

* HTML, CSS, JavaScript (Vanilla)

### Backend

* FastAPI (Python)
* OpenRouter API (LLM integration)

---

## 🧠 How It Works

1. User sends a message
2. Frontend sends request to FastAPI backend
3. Backend processes message using LLM (OpenRouter)
4. AI generates a contextual response
5. Response is displayed in chat UI
6. Mood and session data are tracked per interaction
